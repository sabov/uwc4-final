#  NodeJS Feedparser

We parse your feeds in node.js using
Isaac Schlueter's [sax](https://github.com/isaacs/sax-js) parser.

## Requirements

    "express": "3.0.3",
    "jade": "*",
    "async": "~0.2.7",
    "connect": "~2.7.6",
    "mysql": "~2.0.0-alpha7",
    "xtend": "~2.0.3",
    "forever": "~0.10.0",
    "feedparser": "~0.15.2",
    "request": "~2.16.6"

## Installation

```bash
git clone
npm install
grunt dev||prod
```
